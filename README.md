# MarkDown Study


1. 글머리 기호 
>[숫자]+[.]+[간격] 을 주면 글머리 기호가 된다.

2. 헤더 
>[#]+[간격] 을 주면 글 머리로 구분 됨.

3. BlockQuote
>[>] 문자를 사용해서 한다.

4. Code 삽입
>[```]+[언어타입] code [```]
>```cpp 코드 ``` 이런식으로 한다.

5. 강조 
>*강조 어구*

6. Image
\!\[Alt text\](/path/to/img.jpg)

7. Link <br/>
\[GitHub\]\(http://github.com)<br/>
[구글](http://google.co.kr)
